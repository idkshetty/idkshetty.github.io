GeoCheckout

This project is a web-based address finder using Leaflet.js for maps and Nominatim API for address search and geolocation. Users can search for addresses, use their current location, and confirm the location with latitude and longitude. The interface includes a draggable marker, address suggestions, and clear/reset options, making it ideal for e-commerce or delivery apps.

Advanced Weather Forecast App

The Advanced Weather Forecast App is a sleek and user-friendly web application that provides real-time weather information and a 12-hour hourly forecast. Users can enter a city name to view current weather details like temperature, humidity, wind speed, and weather conditions, along with a forecast for the next 12 hours. Featuring a modern dark mode interface, the app offers an intuitive design with easy navigation, powered by WeatherAPI for accurate data. Perfect for anyone needing reliable weather updates in a visually appealing format.
